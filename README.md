# Powerbi-sales-insights-project

## 🚀 Project Overview

This project focuses on analyzing a retail Superstore dataset to uncover key business insights related to sales, profit, customers, and product performance using Power BI.

---

## 🎯 Objectives

* Analyze sales and profit trends
* Identify top-performing products and regions
* Understand the impact of discounts on profitability
* Detect loss-making segments

---

## 📁 Dataset

* Source: Superstore Dataset
* Rows: 51,000+
* Features: Sales, Profit, Discount, Category, Region, Customer Segment, Shipping Cost

---

## 🛠 Tools & Technologies

* Power BI (Dashboard & Visualization)
* Excel / CSV (Data Cleaning)
* Python (EDA)

---

## 📊 Dashboard Features

* 💰 Total Sales & Profit KPIs
* 🌍 Region-wise Sales Analysis
* 📦 Category & Sub-category Performance
* 📉 Profit vs Discount Analysis
* 🚨 Loss-making Products Identification
